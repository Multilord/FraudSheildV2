# Dataset Directory

Place the IEEE-CIS Fraud Detection dataset files here:

  data/train_transaction.csv   (~530 MB)
  data/train_identity.csv      (~40 MB)

Download from: https://www.kaggle.com/c/ieee-fraud-detection/data

Without the files the platform runs on pre-computed statistics from the full
590,540-row dataset. The ML scoring engine works in both modes.
